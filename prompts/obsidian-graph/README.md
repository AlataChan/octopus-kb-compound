# Obsidian Graph Prompt Pack

Prompt pack for keeping an Obsidian-backed LLM knowledge base readable, linkable, and queryable.

Files:

- `retrieve.md`: retrieval prompt template
- `maintain.md`: maintenance prompt template
- `linking-policy.md`: canonical link and graph hygiene policy

Use the expanded example vault at `examples/expanded-vault/` to test prompt behavior across concept, entity, comparison, timeline, index, log, and raw-source pages before applying the policy to a production vault.
